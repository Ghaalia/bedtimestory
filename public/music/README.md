# Background music tracks

Place royalty-free MP3 files here, named to match `lib/music.ts`:

- `calm.mp3`     — gentle, sleepy
- `adventure.mp3` — light, hopeful
- `magical.mp3`  — twinkly, dreamy
- `silly.mp3`    — playful, bouncy

Suggested sources (verify license before shipping):
- https://freemusicarchive.org/  (filter by CC-BY)
- https://pixabay.com/music/     (royalty-free)
- https://incompetech.com/       (Kevin MacLeod)

The player loops these underneath the narration at ~18% volume.
